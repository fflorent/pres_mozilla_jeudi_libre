# Les devtools
## Les améliorations notables en 2015 * L'outil d'analyse mémoire (enfin !) * De l'aide pour le CSS * Pour les animations, la sélection des couleurs, filtrage, etc. * Des extensions pour les devtools * FireQuery 2.0 (jQuery) * Pixel Perfect (superposition de maquettes) * RDP Inspector (pour les développeurs d'extensions des devtools)
## Ce qui vient en 2016 * Thème Firebug pour les devtools (addon) * Déjà en beta * WebSocket Inspector * Addon en beta, sera inclus dans Firefox * Inclus avec : Parser JSON, Socket.io, Sock.js * D'autres extensions
